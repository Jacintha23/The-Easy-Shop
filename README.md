# The EasyShop: General Goods Store 
## A small web app for selling everyday items. Built with Spring Boot, developed in IntelliJ, and tested with Insomnia.
## What it does:

### * Manage products and categories (create, read, update, delete)

### * Session-based shopping cart

### * Place basic orders

### * Exposes a REST API for all core actions

### * Basic validation and error handling
